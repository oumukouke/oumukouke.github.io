【踩坑日记】Hexo+github搭建的博客无法显示图片

发表于 2020-12-24 更新于 2020-12-25

1 问题描述

当本地MarkDown博客里插入了图片，在将博客推送到github.io上后，原本插入的图片无法显示

2 解决办法

2.1修改 hexo 目录下的 .config.yml 文件

​ 将 post_asset_folder: false 修改为 post_asset_folder: true

2.2安装hexo-asset-image插件。

​ 在hexo目录下打开 Git Bash。输入npm install hexo-asset-image --save

2.3 安装完上述插件后，当我们创建新文章时 ，会自动在这个文章的同目录下创建一个同名的文件夹。

2.4将图片放进这个同名文件夹中，然后在md文件中，使用![](图片名字)语句来插入图片

2.5需要注意的是，这样的引用在md编辑器里面是看不到图片的（我使用的Typora就无法显示），但是图片可以在浏览器上显示
